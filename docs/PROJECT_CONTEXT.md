Project Name: Truck Booking Platform

Goal:
Build a scalable truck booking web application similar to Porter or TruckGuru.

Tech Stack:
Frontend:
- ReactJS
- TailwindCSS

Backend:
- Java
- Spring Boot
- Spring Security
- PostgreSQL

Architecture:
- Modular monolith (initially)
- Event-driven components
- REST APIs

Core Users:
1. Customer
2. Transporter

Core Features:
- Authentication
- Truck listing
- Truck search
- Booking system
- Booking tracking
- Ratings

Future Features:
- Driver management
- Live GPS tracking
- Payment gateway
